前言

  当百度一道题目的答案时，总会出现哪些熟悉的字眼，例如：百度知道、百度文库、问答库、上学吧和考试资料网等等。这其中总有那么一两个收费的，点进去这几个网站多了，也就会发现，原来这几个网站收费不是没有道理，人家的数据库存储的题全啊。既然不能直接获取答案，那么只能间接获取了。
  因此，我对这些方法做一下简单的整理，防止我以后再为一道题的正确答案而发愁。

正文

初阶查法[1]

  • 上学吧查题（清爽界面）

    • 使用方法：直接输入题目即可
    • 注意事项：这个目前来说体验最好，直接输入题目即可得到答案。

  • 优学优考网（有点麻烦）

    • 使用方法：输入题目，关注公众号回复相应数字得到答案，只能在手机上看答案。

  • 注意事项：虽然有点麻烦，但这个网站很稳定，应该是后台开的 VIP 号，但不知道查询多了会不会有限制。

  • 学小易（正规题库，全网课平台）

    • 使用方法：两种方法，一种是关注公众号输入题目得到答案。另一种是下载它的 APP 查题，目前来说可以免费查询。
    • 注意事项：APP 在商店里就可以搜到，这个题库明显是这个平台自己收集的，所以题库更新很快，但提供的网页并不能够查题。

  • 网课答案吧（有点冷门）

    • 使用方法：收录超星、高校邦、智慧树等，按需检索。
    • 注意事项：我是因为在这上面找到了新开设的一门尔雅课才使用的，不过答案排版非常不友好。

  • 助课宝（被一星保护的应用）

    • 使用方法：手机下载 APP，提供课本习题答案
    • 注意事项：被收录的课程是很全的，但更新较慢。

  • 搜答案（不太稳定）

    • 使用方法：与上个网站的使用方法相同，先解析，再返回答案
    • 注意事项：不需要关注公众号，需要进去它的主站，获取暗号。

  • 查题吧（方便，但题库少）

    • 使用方法：直接输入题目即可，有客户端（提取码：40y2）。
    • 注意事项：貌似是问答库的 API 接口，题库没有上一个网站的全。

  • 在公众号里回复查题（功能被砍，有点麻烦）

    • 使用方法：在公众号里发送消息，即可查题。
    • 注意事项：这个是我同学推荐的，我以前刷的超星就是用的它，超星的网课收录很全。不过现在这个公众号改版了，以前直接在网站上输入题目就能得到答案。对于电脑玩家有点麻烦。

  • 上学吧找答案vip破解 BETA版（不太稳定）

    • 使用方法：首先在上学吧中查询，将答案页面的网址放入到该网站解析
    • 注意事项：需要关注公众号，获取查题暗号，为了防止有人用机器自动解析。

  • 百度文库（万能的资料库）

    • 说明：在百度文库中搜索题目即可。百度文库虽然题库收录多，但检索条件实在太过贫乏，所以往往找一道题需要浏览一篇文档或者几十页的 PPT。
    • 这里只给出下载的方法，我是用的嗨文库（收费，我是买的 5.5 元 100 次，用了快一年才用了三十多次），只要是用下载卷的，通过它都可以下载到源文件。以前也用过好几个免费的下载网站，但都被我一一取消收藏了，因为下载很不稳定。

  • 豆丁网（文献下载）

    • 说明：暂时没有好方法，上面的文档都可以在线看，可以使用 QQ 的长截图，也可以参考阿虚同学的方法。

  • 道客巴巴（文献下载）

    • 说明：以前有个冰点下载，但因为作者不维护，已经不能用了。新方法见此贴，但一天只能下一次。

总结

  对于网课查题，课本习题答案，你可以通过上面的方法轻易找到。然而它们只给你一个直观的答案，显示的多是暂无解析。所以对于一些需要真正搞懂的题目，还是自己多动动脑吧。
  大部分网站都需要一些另加操作。其实类似于这样的网站，大部分都是使用的同样的题库（数据库）。然后调用相同的 API 接口，大同小异。因此不稳定很正常，且用且珍惜。

附：正规的查题网站

  • 问答库（每天有免费次数）
  • 上学吧（每天免费获取三次答案，之后收费）
  • 考试资料网（收费）
  • 百度知道（免费，但不适合查网课）
